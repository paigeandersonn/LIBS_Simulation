# Validation

The simulate-and-compare workflow used by the worked examples:
element atomic-data setups (NIST transcriptions with provenance),
experimental-spectrum preprocessing, comparison metrics (correlation
coefficient $R$, rms residual, peak matching), and the
[`ValidationCase`][libssim.validation.workflow.ValidationCase]
pipeline that ties plasma conditions and instrument settings into one
reproducible run.
