# Physics

The local-thermodynamic-equilibrium engine: partition functions
$U(T)$, the Saha ionization balance, Boltzmann level populations,
bound-bound emission/absorption coefficients, Doppler–Stark–Voigt
line profiles, and the free-free + free-bound continuum. Each routine
cites the dissertation equation it implements.
