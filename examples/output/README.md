# Generated validation outputs

Every file under this tree is **regenerated by a script in
`examples/`** — nothing here is hand-made, and only the READMEs are
tracked by git. Each subfolder is one validation case and carries its
own README stating what each figure shows, the key numbers, and the
exact regeneration command.

| Folder | Case | Data | Generated by |
|---|---|---|---|
| `hansen_na_500ns/` | Na I D doublet vs a REAL measured spectrum (Hansen thesis, 500 ns) | `data/experimental/hansen_thesis/` | `validate_sodium_hansen.py`, `..._onion.py`, `digitize_hansen_naD.py` |
| `surrogate/` | Workflow-only demos against clearly-labeled synthetic surrogates (no physics validation) | generated in-run | `validate_sodium.py`, `validate_aluminum.py` |
